HTML

HTML (Hypertext Markup Language) é a linguagem essencial para criar páginas web. Usando tags, o HTML estrutura o conteúdo da página, incluindo texto, imagens e links. É a base para o design web, permitindo a adição de estilos (CSS) e interatividade (JavaScript) para uma experiência completa na web.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação e o design de páginas web escritas em HTML ou XML. Com o CSS, os desenvolvedores podem aplicar estilos visuais, como cores, fontes, espaçamento e posicionamento, a elementos específicos em uma página. Ele oferece uma abordagem eficiente para separar a estrutura do conteúdo HTML da sua apresentação, permitindo uma manutenção mais fácil e uma maior flexibilidade de design

JavaScript

JavaScript (JS) é uma linguagem de programação de alto nível amplamente utilizada para tornar páginas web dinâmicas e interativas. Como uma linguagem de script do lado do cliente, o JS é executado nos navegadores dos usuários, permitindo a manipulação do conteúdo HTML, interação com o usuário, e comunicação com servidores. Com sintaxe simples e flexível, o JS é crucial para o desenvolvimento web moderno, possibilitando a criação de funcionalidades como animações, validações de formulários, requisições assíncronas e muito mais.

React

React é uma biblioteca JavaScript utilizada para construir interfaces de usuário interativas e reativas. Baseada em componentes reutilizáveis, React usa um Virtual DOM para otimizar atualizações e fornece uma sintaxe chamada JSX para facilitar a descrição da interface. Com um fluxo de dados unidirecional e uma abordagem declarativa, React é popular pelo desenvolvimento eficiente de aplicações web dinâmicas e escaláveis.

API Rest

API REST (Representational State Transfer) é uma abordagem arquitetural comum para o desenvolvimento de serviços web. Utilizando padrões HTTP, uma API REST permite a comunicação entre sistemas de maneira eficiente e escalável. As operações (como GET, POST, PUT e DELETE) são mapeadas para ações específicas sobre recursos, proporcionando uma interface uniforme. Os dados são geralmente transferidos em formatos como JSON. APIs REST são amplamente adotadas por sua simplicidade, escalabilidade e interoperabilidade entre diferentes sistemas.
